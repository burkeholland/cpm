Switch models
in one command

cpm is a tiny shell tool that makes it easy to switch between BYOK provider models in GitHub Copilot CLI.

Terminal
$  git clone https://github.com/burkeholland/cpm
$  cd cpm && bash install.sh
>  git clone https://github.com/burkeholland/cpm
>  cd cpm ; .\ install.ps1
~ / project
$ cpm
Pick a model:
  1) MiniMax > MiniMax-M2.7
  2) Ollama > llama3.2
  3) Anthropic > claude-opus-4-5
Enter number (1-3): 1
✓ Switched to MiniMax > MiniMax-M2.7
$ copilot
Features

Everything you need

⚡

Interactive Picker

Numbered menu shows Provider > Model. Pick a number, get your env vars set instantly.

🔑

Key Management

Prompts for missing API keys inline and saves them to your shell profile automatically.

📦

VS Code Import

Already configured BYOK models in VS Code? Import them with cpm import.

🐚

Cross-Shell

Works in Bash, Zsh, and PowerShell. Shell function design means env vars stick in your session.

🔒

No Secrets in Config

Config stores env var names, not keys. Your secrets stay in your shell profile where they belong.

📝

Simple JSON Config

One JSON file at ~/.config/cpm/models.json. Edit it with cpm edit.

Reference

All commands

cpm Interactive model picker
cpm status Show active model & env vars
cpm list List all configured models
cpm keys Show & set API keys
cpm edit Open config in $EDITOR
cpm import Import from VS Code
cpm clear Unset all provider env vars
cpm help Show help
